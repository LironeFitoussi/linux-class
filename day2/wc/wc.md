# 🟦 הסבר מקצועי על הפקודה wc

## 🔹 מה זה wc?

`wc` = **Word Count**
פקודה שמאפשרת לספור בקלות:

* כמה **שורות** יש בקובץ
* כמה **מילים**
* כמה **תווים / bytes**
* או הכול ביחד — בפורמט מסודר

זה כלי בסיסי בלינוקס, נפוץ מאוד בניתוח טקסט, לוגים ומסמכים.

---

# 🛠️ איך עובדים עם wc

אתה מריץ את `wc` על קובץ — והוא נותן לך נתונים מדויקים.

## 🎯 4 הדברים המרכזיים שהוא מסוגל לספור:

### 1️⃣ ספירת שורות

```
wc -l notes.txt
```

מחזיר את מספר השורות (כולל שורות ריקות).

---

### 2️⃣ ספירת מילים

```
wc -w notes.txt
```

סופר כל מילה — מוגדר כל רצף תווים שמופרד ברווח.

---

### 3️⃣ ספירת תווים

```
wc -c notes.txt
```

סופר **bytes**, כלומר אורך קובץ.
(במערכות UTF-8, תווים מסוימים תופסים יותר מ־1 byte.)

---

### 4️⃣ לקבל הכול ביחד

```
wc notes.txt
```

הפלט ייראה כך:

```
  6  21  145 notes.txt
```

המשמעות:

* 6 שורות
* 21 מילים
* 145 bytes

---

# 🎯 למה wc חשוב?

* בדיקות מהירות של גודל קובץ
* הערכה של נפח לוגים
* עיבוד טקסטים בתהליכי DevOps
* שילוב בצינורות (pipes) עם grep / sort / awk
* כתיבת סקריפטים בלינוקס

`wc` הוא כלי דיאגנוסטי קטן ופשוט — אבל בשימוש יומיומי מסיבי.
