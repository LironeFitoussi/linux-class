# 🟦 הסבר מלא על uniq

## 🔹 מה זה uniq?

`uniq` הוא כלי שמסנן *שורות כפולות* מתוך קובץ.
הוא לא מוחק שורות — אלא מציג את הקובץ אחרי הסרת כפילויות.

בפועל, uniq משמש ל:

* ניקוי רשימות
* בדיקה אם יש כפילויות
* ספירת שכיחויות
* בניית pipelines של ניתוח נתונים

---

# 🛠️ איך עובדים עם uniq

## 1️⃣ הקובץ חייב להיות **ממויין** או שהכפילויות יהיו **צמודות**

למשל, זה יעבוד:

```
orange
orange
orange
```

אבל זה **לא** יתפס ככפילות:

```
orange
banana
orange
```

לכן תמיד בשימוש מקצועי עושים:

```
sort items.txt | uniq
```

---

# 🟩 דוגמאות לפי הבקשה שלך

## 🔹 **הסרת כפילויות**

```
uniq items.txt
```

מציג רק מופע אחד מכל שורה כפולה — אבל *רק אם הן רצופות*.

פלט לדוגמה:

```
apple
banana
orange
pear
kiwi
apple
```

---

## 🔹 **ספירת כפילויות**

```
uniq -c items.txt
```

פלט לדוגמה:

```
  2 apple
  1 banana
  3 orange
  1 pear
  2 kiwi
  2 apple
```

---

## 🔹 **שימוש נכון: יחד עם sort**

```
sort items.txt | uniq -c
```

זה הפייפליין המקצועי:

* sort → ממיין את הקובץ
* uniq -c → סופר כמה פעמים כל שורה מופיעה

פלט לדוגמה:

```
  4 apple
  1 banana
  3 orange
  1 pear
  2 kiwi
```

---

# 🎯 למה uniq חשוב?

* קריטי בניתוח לוגים
* מאפשר לראות שכיחויות של אירועים
* משמש לניקוי דאטה
* בונה בסיס להמשך עבודה עם awk/sed
* משמש בכל תהליכי DevOps/IT של בדיקת עומסים, חיבורים, אירועים חוזרים ועוד
